Gerenciamento de Banco de Dados
SHOW DATABASES;Exibe uma lista de todas as bases de dados no servidor MySQL.- Permite visualizar todas as bases de dados presentes no servidor.
SHOW TABLES;Mostra as tabelas em um banco de dados específico.- Exige a especificação do nome do banco de dados no qual as tabelas serão listadas.
SHOW COLUMNS FROM nome_da_tabela;Exibe os nomes e tipos de colunas de uma tabela específica.- Substitua nome_da_tabela pelo nome real da tabela que deseja consultar.
Detalhes das Colunas
SHOW INDEX FROM nome_da_tabela;Mostra informações sobre os índices de uma tabela.
SHOW FULL COLUMNS FROM nome_da_tabela;Exibe informações detalhadas sobre as colunas de uma tabela, incluindo detalhes como tipo de dado, padrão, nulo, chave primária, etc.
DESC nome_da_tabela;Descreve a estrutura de uma tabela, mostrando os nomes, tipos e outras propriedades das colunas.
Criação de Objetos de Banco de Dados
SHOW CREATE TABLE nome_da_tabela;Exibe o código SQL usado na criação de uma tabela específica.
SHOW CREATE PROCEDURE nome_do_procedimento;Exibe o código SQL usado na criação de um procedimento armazenado específico.
SHOW CREATE FUNCTION nome_da_função;Exibe o código SQL usado na criação de uma função específica.
Privilégios de UsuárioSHOW GRANTS FOR nome_do_usuário@localhost;Mostra os privilégios de acesso aos bancos de dados para um usuário específico.
Ferramenta de Linha de Comandomysqlshow -u usuário -p [banco [tabela [coluna]]];Opera diretamente no shell do Linux e permite obter informações sobre bancos de dados, tabelas e colunas.
Administração do Servidor
SHOW CHARACTER SETS;Lista os conjuntos de caracteres disponíveis no servidor MySQL.- Apresenta a lista de conjuntos de caracteres que podem ser utilizados no servidor.
SHOW COLLATIONS;Mostra as colações disponíveis no servidor MySQL.- Exibe as colações configuradas para os conjuntos de caracteres, definindo regras de comparação.
SHOW ENGINES;Exibe os mecanismos de armazenamento disponíveis no servidor MySQL.- Apresenta os mecanismos de armazenamento de dados disponíveis para uso nas tabelas.
SHOW EVENTS;Lista os eventos agendados no servidor MySQL.- Mostra eventos agendados para serem executados automaticamente em horários específicos.
SHOW FUNCTION STATUS;Apresenta informações sobre o status das funções armazenadas no servidor MySQL.- Fornece informações sobre o uso e o desempenho das funções armazenadas.
SHOW GLOBAL VARIABLES;Exibe as variáveis ??globais do servidor MySQL.- Mostra as variáveis ??com configurações que afetam o comportamento geral do servidor.
SHOW INNODB STATUS;Apresenta informações detalhadas sobre o status do engine de armazenamento InnoDB.- Fornece informações detalhadas sobre o funcionamento do mecanismo de armazenamento InnoDB.
SHOW MASTER LOGS;Lista os logs mestres do MySQL Replication.- Exibe informações sobre os logs binários do MySQL Replication. Esses logs registram todas as alterações feitas no banco de dados master para que possam ser replicadas em servidores slave.
SHOW OPEN TASKS;Mostra as tarefas em execução no servidor MySQL.- Exibe as tarefas ativas no servidor, como consultas, transações e processos de limpeza.
SHOW PERFORMANCE SCHEMA;Exibe informações sobre o desempenho do servidor MySQL.- Apresenta métricas detalhadas sobre o desempenho do servidor, como uso de CPU, memória e E/S.
SHOW PLUGINS;Lista os plugins carregados no servidor MySQL.- Mostra os plugins de terceiros instalados e ativos no servidor.
SHOW PROCESSLIST;Exibe a lista de processos em execução no servidor MySQL.- Apresenta informações sobre todos os processos em execução no servidor, incluindo consultas, transações e threads de gerenciamento.
SHOW PROFILES;Mostra os perfis de armazenamento configurados no servidor MySQL.- Exibe os perfis de armazenamento definidos para otimizar o desempenho de consultas em tabelas específicas.
SHOW REPLICATION SLAVE STATUS;Apresenta informações sobre o status do slave no MySQL Replication.- Fornece detalhes sobre o status da replicação no servidor slave, incluindo atraso e progresso da replicação.
SHOW SESSION STATUS;Exibe informações sobre o status da sessão atual do MySQL.- Apresenta informações sobre variáveis ??de sessão e configurações que afetam a consulta atual.
SHOW SLAVE STATUS;Mostra o status do slave no MySQL Replication.- Exibe o status da replicação no servidor slave, incluindo atraso e progresso da replicação.
SHOW STORAGE ENGINES;Lista os mecanismos de armazenamento disponíveis no servidor MySQL.- Apresenta os mecanismos de armazenamento de dados disponíveis para uso nas tabelas.
SHOW VARIABLES;Exibe as variáveis ??do sistema e de sessão do servidor MySQL.- Mostra as variáveis ??com configurações que afetam o comportamento do servidor e das sessões de usuário.
Comandos Adicionais
SHOW BINARY LOGS;Mostra os logs binários do MySQL Replication.- Exibe os logs binários que registram todas as alterações feitas no banco de dados master para fins de replicação.
SHOW MASTER STATUS GTID;Exibe informações detalhadas sobre o GTID (Global Transaction Identifier) do master no MySQL Replication.- Apresenta informações sobre o GTID do servidor master, utilizado para identificar transações na replicação.
SHOW WARNINGS;Apresenta os avisos gerados na consulta anterior.- Exibe avisos que podem indicar problemas potenciais com a consulta executada anteriormente.
