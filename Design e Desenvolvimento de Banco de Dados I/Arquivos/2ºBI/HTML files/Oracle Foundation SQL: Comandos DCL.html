GRANT

Concede permissões específicas a usuários ou papéis sobre objetos do banco de dados.

GRANT SELECT ON tabela TO usuario;

Imagine que você é o proprietário de uma biblioteca e concede permissão para alguém consultar um livro específico.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

REVOKE

Revoga as permissões concedidas anteriormente a usuários ou papéis sobre objetos do banco de dados.

REVOKE INSERT ON tabela FROM usuario;

Suponha que você tenha emprestado um livro para alguém e, posteriormente, decida retirar esse privilégio de empréstimo.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

DENY

Nega explicitamente permissões a usuários ou roles em objetos do banco de dados.

DENY DELETE ON tabela_exemplo TO usuario3;

Imagine que você deseja bloquear permanentemente o acesso de alguém à sua casa, independentemente de qualquer outra coisa.

SQL Server

CREATE USER

Cria um novo usuário no banco de dados.

CREATE USER novo_usuario IDENTIFIED BY 'senha';

Imagine que você está organizando uma festa e convida um novo amigo para participar.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

DROP USER

Remove um usuário existente do banco de dados.

DROP USER usuario_antigo;

Imagine que um dos seus amigos não poderá mais comparecer à festa, então você decide retirá-lo da lista de convidados.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

ALTER USER

Modifica as propriedades de um usuário existente no banco de dados.

ALTER USER usuario_atual RENAME TO novo_nome;

Imagine que um dos seus amigos quer mudar o nome pelo qual é chamado.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

CREATE ROLE

Cria uma nova role (grupo de permissões) no banco de dados.

CREATE ROLE novo_grupo;

Imagine que você está formando uma equipe de trabalho e cria um novo grupo para atribuir tarefas específicas.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

DROP ROLE

Remove uma role existente do banco de dados.

DROP ROLE grupo_antigo;

Imagine que uma equipe anterior já não é mais necessária, então você dissolve o grupo.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

ALTER ROLE

Modifica as propriedades de uma role existente no banco de dados.

ALTER ROLE grupo_atual RENAME TO novo_nome;

Imagine que você quer renomear um departamento da sua empresa para refletir uma nova estrutura organizacional.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite

SET ROLE

Define o papel ativo para a sessão atual.

SET ROLE novo_grupo;

Imagine que você está assumindo um papel diferente dentro da sua equipe, com responsabilidades distintas.

PostgreSQL, Oracle DB, SQL Server

SHOW GRANTS

Mostra as permissões concedidas a um usuário específico.

SHOW GRANTS FOR usuario1;

Imagine que você quer verificar quais são as permissões concedidas a um amigo específico para entrar na sua casa.

MySQL, PostgreSQL, Oracle DB, SQL Server, SQLite
